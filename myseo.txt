      <h2>About This Page</h2>
      <p>
        This web page demonstrates the use of semantic HTML elements to create a clear, accessible, and search engine optimized layout. It avoids unnecessary <code>&lt;div&gt;</code> and <code>&lt;span&gt;</code> elements, focusing on meaningful tags that describe the content structure.
      </p>
    </section>

    <section id="principles">
      <h2>Core Principles of Semantic Web Design</h2>
      <article>
        <h3>Human Readability</h3>
        <p>
          Content is organized with clear headings, descriptive sections, and logical flow, making it easy for users to read and navigate.
        </p>
      </article>
      <article>
        <h3>Machine Readability</h3>
        <p>
          Semantic tags like <code>&lt;header&gt;</code>, <code>&lt;nav&gt;</code>, <code>&lt;main&gt;</code>, <code>&lt;section&gt;</code>, and <code>&lt;footer&gt;</code> help search engines and assistive technologies understand the structure and purpose of the content.
        </p>
      </article>
      <article>
        <h3>SEO Best Practices</h3>
        <ul>
          <li>Use of heading tags (<code>&lt;h1&gt;</code> to <code>&lt;h3&gt;</code>) in correct order</li>
          <li>Descriptive <code>&lt;title&gt;</code> and <code>&lt;meta name="description"&gt;</code></li>
          <li>Clear navigation with accessible links</li>
          <li>Descriptive, keyword-rich content</li>
        </ul>
      </article>
